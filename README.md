### make sure your picture's pixel under 1000*1000
### use ascii.json to set what you want to convert
### put your pictures in data directory

### support format(C# bitmap)
- `*.jpg`
- `*.jpeg`
- `*.jfif`
- `*.png`
- `*.gif`
- `*.bmp`
